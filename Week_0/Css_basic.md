### Common styling attributes

```
color -> Color property are use to color the text.

background -> Background are use to put background in our page.

border-radius -> The border-radius property in CSS is used to create rounded corners for elements.

border -> The border property creates a border around an HTML element, such as a div, image, or text.

padding -> Padding is the space between an element's content and its border. 

margin -> Margin is the space outside an element, separating it from other elements.

box-shadow -> Box shadow property are use to show shadow of button text etc.
```
