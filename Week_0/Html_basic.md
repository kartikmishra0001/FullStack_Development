
### HTML Tags 
```
<html> 
An HTML tag is a piece of markup language used to indicate the beginning and end of an HTML element in an HTML document.

<head>
The <head> tag in HTML is a container for metadata about the document.

<title>
The <title> tag in HTML we wrote our titles.

<body>
The <body> tag in HTML defines the document's body and contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

<div>
The <div> tag in HTML is a block-level element used to define a division or section within a document. 

<span>
The HTML <span> tag is an inline container used to group and style text or elements within a document. 

<h1>...<h6>
The HTML <h1>...<h6> are used to write headings.

<p>
The HTML <p> are used to write Paragraphs.

<img>
The HTML <img> are used to show Images in our page.

<a>
The HTML <a> tag, also known as the anchor tag, is used to create hyperlinks. 

<input>
The <input> tag in HTML is used to create interactive controls for web-based forms to accept data from the user. 

<button>
The <button> tag in HTML is used to create clickable buttons.

<b>/<i>
The <b> and <i> tags in HTML are both used to style text in italic and bold.

<center>
To center-align content in HTML, you can use CSS properties. The <center> tag is deprecated in HTML5, so it's recommended to use CSS for centering content.
```
