# What is Dom ?

The DOM, or Document Object Model, is a programming interface for web documents. It represents the structure of a web page as a tree of objects.

## Static HTML 

As the name suggests, static HTML represents HTML that does not change.

## Dynamic HTML 

How can you update the elements of the page dynamically?

### Document object -

 In the browser, the document object is a fundamental part of the Document Object Model (DOM). It represents the web page currently loaded in the browser and provides a way to interact with and manipulate its content.

## Fetching element 

There are 5 popular methods available for fetching DOM elements - 
1. querySelector
2. querySelectorAll
3. getElementById
4. getElementByClassName
5. getElementsByClassName

## Updating element 

1. innerHTML - Used for updating the HTML inside an element
2. textContent - Used for updating the text content inside an element

## Deleting element 

1. removeChild - Removes a specific node of a parent
2. onclick - function that triggers whenever you click on a button
